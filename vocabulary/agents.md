# Vocabulary Agent Notes

Purpose: use this folder to improve CELPIP speaking and writing vocabulary efficiently.

## File Usage
- `notes.md`: teacher-course notes. Prefer this file for usage patterns, sentence upgrades, connectors, punctuation, and example style.
- `400_words.md`: core shared vocabulary for both speaking and writing practice.
- `200_vocab.md`: extended vocabulary list with meanings and examples.
- `200_phrasal_verbs.md`: phrasal verbs for natural speaking and informal/semi-formal writing where appropriate.

## How To Use These Files
- When teaching or correcting word choice, check `notes.md` first for better usage and example patterns.
- Treat `400_words.md` as a high-priority study list for active recall, drills, and sentence building in both speaking and writing.
- If the user asks for practice from the `400_words` list, work in manageable chunks such as `20` or `25` words at a time.
- Prefer natural, accurate usage over forced “advanced” vocabulary.
- Give CELPIP-friendly examples: practical, everyday, workplace, community, service, housing, study, and opinion topics.

## Output Preference
- For vocabulary help, include meaning, a natural example, and when useful, a safer alternative or common collocation.
- When upgrading a sentence, avoid making it sound unnatural or overly academic.
